# Same Bid

## 📝 Description :
#### Go through a linked list to find a node.
---
## Parameters :
- head: head of the linked list.
- id: id of the node to find.
---
## Return :
- Return 1 if the node has been found, and 0 if not.
---
## Proto :
```c
int same_nid(t_list* head, int id);
```
---
## Exemple : 
```
head = [2]->[3]->NULL
```

```c
same_nid(head, 3);
```

```$> 1```
