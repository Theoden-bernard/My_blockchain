# Free Block List

## 📝 Description :
#### Free every element of a block list.
---
## Parameters :
- head_block: head of the block list.
---
## Return :
- This function doesn't return anything
---
## Proto :
```c
void free_block_list(t_block* head_block);
```
---
## Exemple : 
```c
free_block_list(head_block);
```

```$> No return```
