# Print Block List

## 📝 Description :
#### Print every block id of a linked list
---
## Parameters :
- head: head of the block list to print.
---
## Return :
- This function doesn't return anything.
---
## Proto :
```c
void print_double_list(t_block* head);
```
---
## Exemple : 
```c
print_double_list(head);
```

```$> No return```
